# RSS Feed for LinkedIn Auto-posting via Zapier - Spec document

---
Status: done
Last updated: 2026-03-15
---

## Purpose
Add an `/rss.xml` route to the blog that generates a valid RSS 2.0 feed from all published notes, consumed by a Zapier Zap to automatically create LinkedIn posts when new notes are published.

## Objectives
1. Generate a valid RSS 2.0 feed at `/rss.xml` from all published notes
2. Support per-post customisation of the LinkedIn post body via a `socialPost` frontmatter field
3. Automate LinkedIn posting via Zapier with no manual steps after deploy

## Scope

### In scope
- `GET` route handler at `src/app/rss.xml/route.ts` returning RSS 2.0 XML
- Optional `socialPost` frontmatter field on notes
- Static generation of the feed at build time

### Out of scope
- Recipes are not included in the RSS feed (notes only)
- Zapier setup is a manual step (see implementation plan)

## Requirements

### Functional requirements
1. Given a note has a `socialPost` frontmatter field, when the RSS feed is generated, then `<description>` uses `socialPost` instead of `summary`
2. Given a note has no `socialPost` field, when the RSS feed is generated, then `<description>` falls back to `summary`
3. Given the site is built, when `/rss.xml` is requested, then a valid RSS 2.0 document is returned with `Content-Type: application/xml`
4. Given the site is built, when `/rss.xml` is requested, then each `<item>` includes `<title>`, `<description>`, `<link>`, `<pubDate>`, and `<guid>`

### Non-functional requirements
1. Given the site is built, when the feed is served, then it is statically generated (no dynamic server rendering)

## Architecture overview
The feed is implemented as a Next.js Route Handler (`route.ts`) that calls `getPosts()` at build time and serialises the result to RSS 2.0 XML. Static generation is achieved via `export const dynamic = 'force-static'`.

## Tools & Technology
- Next.js Route Handlers (App Router)
- `getPosts()` utility (`src/functions/getPosts.ts`)
- `gray-matter` frontmatter parsing (existing)
